# Machine learning-based surrogate for 2D mechanics problems

Python code for solving a 2D boundary value problem using a machine learning-based (neural) surrogate constitutive model.

First, the neural constitutive model needs to be trained 
Step 1: unzip src/neural-constitutive-model/data.zip
Step 2: run src/neural-constitutive-model/train_constitutive_model.py

Then, run the ml-2d.ipynb notebook to carry out the simulation.

## Prerequisites
- Python 3.10
- Numpy 1.21.2
- Matplotlib 3.4.3
- Pytorch 2.0.1
